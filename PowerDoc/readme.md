# PowerDoc Remediation script

### Actions
- Terminates processes associated with PowerDoc Adware
- Remove PowerDoc Adware from the file system.
- Remove PowerDoc Adware registry keys.
- Remove PowerDoc uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove PowerDoc. The script is intended for CrowdStrike but can be run without it.
