# CrystalPDF Remediation script

### Actions
- Terminates processes associated with CrystalPDF Adware
- Remove CrystalPDF Adware from the file system.
- Remove CrystalPDF Adware registry keys.
- Remove CrystalPDF uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove CrystalPDF. The script is intended for CrowdStrike but can be run without it.
