# PDFTool Adware Remediation script

### Actions
- Terminates processes associated with PDFTool Adware
- Remove PDFTool Adware from the file system.
- Removes PDFTool Adware uninstall key.
- Removes PDFTool Adware registry keys.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description

A remediation script was created for PDFTool thanks to @BradleyOutland
 for reporting the issue https://github.com/xephora/Threat-Remediation-Scripts/issues/19. The script is intended for CrowdStrike but can be run without it.
