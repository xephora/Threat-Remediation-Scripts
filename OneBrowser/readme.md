# OneBrowser Adware Remediation script

### Actions
- Terminates processes associated with OneBrowser Adware
- Remove OneBrowser Adware from the file system.
- Removes OneBrowser Adware uninstall key.
- Removes OneBrowser Adware registry keys.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description

A remediation script was created for OneBrowser thanks to @BradleyOutland
 for reporting the issue https://github.com/xephora/Threat-Remediation-Scripts/issues/19. The script is intended for CrowdStrike but can be run without it.
