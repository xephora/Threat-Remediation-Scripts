# RemotePC Remediation script

### Actions
- Terminates processes associated with RemotePC unwanted software
- Remove RemotePC services.
- Remove RemotePC scheduled tasks.
- Remove RemotePC unwanted software from the file system.
- Remove RemotePC unwanted software registry keys.
- Remove RemotePC uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove RemotePC. The script is intended for CrowdStrike but can be run without it.
