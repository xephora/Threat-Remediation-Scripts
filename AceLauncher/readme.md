# AceLauncher Remediation script

### Actions
- Terminates processes associated with AceLauncher Adware
- Remove AceLauncher Adware from the file system.
- Remove AceLauncher Adware registry keys.
- Remove AceLauncher uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove AceLauncher. The script is intended for CrowdStrike but can be run without it.
