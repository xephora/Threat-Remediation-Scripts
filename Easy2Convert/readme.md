# Easy2Convert Adware Remediation script

### Actions
- Terminates processes associated with Easy2Convert Adware
- Remove Easy2Convert Adware from the file system.
- Removes Easy2Convert Adware uninstall key.
- Removes Easy2Convert Adware registry keys.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description

A remediation script was created for Easy2Convert thanks to @BradleyOutland
 for reporting the issue https://github.com/xephora/Threat-Remediation-Scripts/issues/19. The script is intended for CrowdStrike but can be run without it.
