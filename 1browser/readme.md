# 1browser Remediation script

### Actions
- Terminates processes associated with 1browser unwanted software
- Remove 1browser unwanted software from the file system.
- Remove 1browser unwanted software registry keys.
- Remove 1browser uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove 1browser. The script is intended for CrowdStrike but can be run without it.
